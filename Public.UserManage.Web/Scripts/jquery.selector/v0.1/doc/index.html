该插件数据包括三个部分：索引（目前不能自定义）、分类、列表项。分类信息可以省略，当包含分类信息时，会按照分类的首写字母进行索引；当不包含分类时会按照列表项的首字母进行索引。

  3. </div>

方法/属性说明:

名称说明
getSelected(int:id)获取到当前所选中的对象数组，数组中对象的属性与传入的item json属性相同
checked(int:id)设置选中一个对象
unchecked(int:id)取消选中一个对象
close()关闭显示对话框
dispose()释放资源，在不需要该控件的时候通过该方法释放数据
